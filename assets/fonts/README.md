# Arabic PDF Font

Place `Amiri-Regular.ttf` in this directory for Arabic PDF tutor exports.

Download from: https://fonts.google.com/specimen/Amiri

Or run:
```powershell
Invoke-WebRequest -Uri "https://github.com/aliftype/amiri/raw/master/Amiri-Regular.ttf" -OutFile "Amiri-Regular.ttf"
```

Set `ARABIC_PDF_FONT_PATH=assets/fonts/Amiri-Regular.ttf` in `.env` if using a custom path.
